# AR15.build Feedback & Data Reports

[![Report an Issue](https://img.shields.io/badge/Report-an%20Issue-blue?style=for-the-badge)](../../issues/new/choose)

This repository is the public feedback tracker for AR15.build.

AR15.build is a configuration and comparison platform designed to help users research components, evaluate compatibility, and make informed purchasing decisions across multiple retailers.

We believe transparency and accuracy are critical. This repository exists so users, builders, and industry partners can report issues and suggest improvements directly.

## What You Can Report

Please use the appropriate issue form for:

- Incorrect product specifications
- Compatibility errors or mismatches
- Broken links or pricing inconsistencies
- Guide corrections or clarification requests
- Feature suggestions
- Data feed anomalies

All reports are reviewed and triaged.

## How to File Feedback

When you click **New issue**, GitHub will present the available forms.
You can also review the templates directly:

- [Incorrect Product Specifications](.github/ISSUE_TEMPLATE/incorrect-product-specifications.yml)
- [Compatibility Error or Mismatch](.github/ISSUE_TEMPLATE/compatibility-errors-or-mismatches.yml)
- [Broken Link or Pricing Inconsistency](.github/ISSUE_TEMPLATE/broken-links-or-pricing-inconsistencies.yml)
- [Guide Correction or Clarification](.github/ISSUE_TEMPLATE/guide-corrections-or-clarifications.yml)
- [Feature Suggestion](.github/ISSUE_TEMPLATE/feature-suggestion.yml)
- [Data Feed Anomaly](.github/ISSUE_TEMPLATE/data-feed-anomaly.yml)

## What This Repository Is Not For

This repository does not provide:

- Firearm assembly instructions
- Legal advice
- Technical gunsmithing support
- Product warranty support

For manufacturer-specific questions, please contact the brand directly.

## Why We Keep This Public

- Transparency builds trust.
- Data accuracy improves the entire ecosystem.
- Community input helps us refine compatibility logic and technical guides.
- Vendors can see and respond to data issues affecting their listings.

## Our Goal

We aim to create a reliable, user-driven resource that empowers builders to make informed decisions while fostering a collaborative environment for continuous improvement. Your feedback is essential to achieving this goal.
